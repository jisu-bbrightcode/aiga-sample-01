## Task 4: ContentThemeService — CRUD + prompt template resolution

**Files:**
- Create: `packages/features/ai-image/service/content-theme.service.ts`

**Step 1: Implement ContentThemeService**

Follow the exact same pattern as `StyleTemplateService` (same file in service/). Key methods:
- `findActive()` — list active themes sorted by sortOrder ASC
- `findById(id)` — single lookup, throw NotFoundException if missing
- `create(input)` — with slug generation + duplicate check
- `update(id, data)` — existence check + update + return fresh
- `delete(id)` — existence check + hard delete
- `resolveThemePrompt(promptTemplate, themeVariables)` — replace `{{key}}` with values, validate all required variables are provided

The `resolveThemePrompt` method:
```typescript
resolveThemePrompt(promptTemplate: string, variables?: Record<string, string>): string {
  const requiredVars = promptTemplate.match(/\{\{(\w+)\}\}/g)?.map(m => m.slice(2, -2)) ?? [];
  const vars = variables ?? {};
  for (const key of requiredVars) {
    if (!vars[key]?.trim()) {
      throw new BadRequestException(`{{${key}}}을(를) 입력해주세요`);
    }
  }
  return requiredVars.reduce((result, key) => result.replace(`{{${key}}}`, vars[key]), promptTemplate);
}
```
