펼치기 트리거는?

A

클릭으로 토글

노드 클릭 = 펼치기. 다시 클릭 또는 바깥 클릭 = 접기. 명시적.

B

호버로 펼치기

마우스 올리면 펼침. 벗어나면 접힘. 빠르게 훑기 좋음.

C

호버 미리보기 + 클릭 고정

호버 = 잠깐 펼침. 클릭 = 고정. 다시 클릭 = 접기. 두 장점 결합.
