태그 시스템 — 어떤 종류의 태그가 노드에 붙는가?

작가가 씬/노드에 붙이는 메타데이터. 필터링, 검색, 시각 구분에 사용.

태그 카테고리별 예시
스토리 구조
분기점 컷씬 엔딩 합류점 전투
감정
긴장 슬픔 유머 공포 감동 분노
캐릭터
엘리아 카이런 마르쿠스
커스텀 (작가가 직접 만듦)
검토 필요 보이스 녹음 현지화 대상

카드에 태그를 어떻게 표시할까?

접힌 상태(C)에서는 공간이 제한적

A

접힌: 최대 2개 + 숫자 / 펼친: 전부

접힌 상태에서 태그 2개만 보이고 "+3" 표시. 펼치면 전체.

B

접힌: 색상 도트만 / 펼친: 태그 텍스트

접힌 상태에서는 카테고리별 색상 도트 3~4개. 펼치면 텍스트 태그.

C

접힌: 태그 안 보임 / 펼친: 전부

접힌 상태는 제목 + 메타만. 태그는 펼쳤을 때만.
