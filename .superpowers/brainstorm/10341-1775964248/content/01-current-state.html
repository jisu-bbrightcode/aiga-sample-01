문제점 / 개선 여지

1. UI 완성도
shadcn 기본 컴포넌트로 급조. 디자인 시스템 미반영. Flotter 톤 부재
2. 번역 에디터 UX
원문↔번역 병렬 편집 경험 부족. 맥락 없이 텍스트만 보임
3. 작업 흐름 불명확
번역가가 "다음에 뭘 해야 하지?"를 알 수 없음. 우선순위/필터 부족
4. AI 번역 통합
AI 번역 버튼은 있으나 실제 동작 없음. UX 미설계

이번 디자인의 초점은?

어느 부분에 집중해서 디자인할까요?

A

전체 리디자인 (Full Redesign)

오버뷰 + 번역 목록 + 에디터 + 사전 — 모든 화면을 Flotter 디자인 시스템에 맞춰 재설계. 가장 큰 범위.

B

번역 에디터 집중 (Editor-First)

번역가가 실제로 작업하는 에디터 화면에 집중. 원문↔번역 병렬 편집, 맥락 패널, AI 어시스트. 핵심 UX만 깊게.

C

대시보드 + 워크플로우 (PM 관점)

현지화 PM이 진행률을 추적하고 AI 번역을 관리하는 대시보드 중심. 번역가 배정, 마감일, 품질 지표 등.

D
