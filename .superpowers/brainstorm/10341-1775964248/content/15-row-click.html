번역 행 클릭 시 — 편집 방식

캔버스/세계관 리스트에서 번역 행을 클릭하면 어떻게 편집할까요?

A

인라인 편집 (리스트 내에서)

행을 클릭하면 해당 셀이 편집 모드로 전환. 페이지 이동 없이 바로 수정. 스프레드시트 느낌. 빠르지만 긴 텍스트에 불편.

B

사이드 패널 (슬라이드 오버)

행 클릭 → 우측에서 패널 슬라이드. 원문(읽기 전용) + 번역(편집) + 용어 사전 참조. 리스트는 뒤에 유지. 맥락 유지 + 편집 공간 확보.

C

별도 에디터 페이지

행 클릭 → 전용 편집 페이지로 이동. 원문↔번역 좌우 분할. 가장 넓은 편집 공간. 하지만 페이지 이동 = 흐름 끊김.

D

행 확장 (Expandable Row)

행 클릭 → 아래로 펼쳐짐. 원문 전체 + 번역 편집 영역이 행 아래에 표시. 아코디언 패턴. 맥락 유지 + 추가 공간.
