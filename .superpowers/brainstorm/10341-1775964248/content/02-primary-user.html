A

작가 중심 (1인 작가가 직접)

작가 본인이 AI로 번역하고 검수. 단순한 워크플로우, "번역 시작" 한 번이면 끝. 전문 도구 느낌보다 창작 도구 느낌.

B

번역가 중심 (전문가 협업)

전문 번역가가 작업할 수 있는 CAT 도구 수준의 에디터. 용어집, 번역 메모리, 맥락 패널. 팀 협업 워크플로우.

C

둘 다 (작가 + 번역가 하이브리드)

작가는 AI로 초벌 번역 → 번역가가 검수/수정. 두 페르소나 모두 지원하되, 작가 경험이 1순위.
